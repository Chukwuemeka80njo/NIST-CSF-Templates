The **Governance** folder contains templates that support the leadership, oversight, and strategic decision-making required for a mature cybersecurity program. These documents align with the governance expectations in the **NIST Cybersecurity Framework (CSF)** and broader GRC standards.

Strong governance ensures that cybersecurity responsibilities, policies, risk decisions, and accountability structures are clearly defined and consistently applied across the organization.
